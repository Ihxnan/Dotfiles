# Dotfiles

- A collection of my personal configuration files and dotfiles for system setup and tool preferences.

---

### System Requirements

- npm
- nodejs
- curl
- git

---

### Installation

please download [git](https://git-scm.com/) first.


#### arch
```
sudo pacman -Syyu
sudo pacman -S git
```
#### debian
```
sudo apt update
sudo apt install git
```
